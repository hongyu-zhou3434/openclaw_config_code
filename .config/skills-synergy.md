## 协同流程详解

### 流程 1: 正常任务执行

```
任务启动
    ↓
任务监控开始（多维度）
    ↓
任务正常执行
    ↓
任务完成检查
    ↓
输出结果验证
    ↓
[完成] → 返回结果
```

### 流程 2: 异常检测与处理

```
任务监控检测到异常
    ↓
self-improving-agent 记录
    ├── 命令失败 → ERRORS.md
    ├── 用户纠正 → LEARNINGS.md
    ├── 能力缺失 → FEATURE_REQUESTS.md
    ↓
分析异常类型
    ├── 能力缺失 ───────┐
    │                   ▼
    │           Find-Skills 搜索
    │                   │
    │           ┌───────┴───────┐
    │           ▼               ▼
    │    [找到技能]      [未找到技能]
    │           │               │
    │           ▼               ▼
    │    安装新技能      记录功能请求
    │    重新执行任务          │
    │                          ▼
    │                   等待后续处理
    │
    ├── 文档理解困难 ────┐
    │                   ▼
    │           Summarize 总结
    │                   │
    │                   ▼
    │           提炼关键信息
    │           辅助决策
    │
    └── 流程问题 ────────┐
                        ▼
                优化策略
                更新系统配置
```

### 流程 3: 持续改进

```
.learnings/ 积累足够数据
    ↓
定期回顾分析
    ↓
识别改进点
    ├── 行为模式 → 更新 SOUL.md
    ├── 工作流程 → 更新 AGENTS.md
    ├── 工具使用 → 更新 TOOLS.md
    └── 知识积累 → 更新 MEMORY.md
    ↓
应用改进
    ↓
验证效果
    ↓
[持续改进闭环]
```
